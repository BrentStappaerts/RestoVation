--om te beginnen 
>git clone *repository link* -->zonder de sterretjes

--Een branch aanmaken om in te werken
>git branch naam

>git checkout naam --> om in u branch te beginnen werken.

--wijzigingen doorvoeren
>git status --> om te kijken wat er verandert is

>git add *bestandsnaam* -->zonder de sterretjes

>git commit -m "tussen deze haakjes de commitmessage met wat je verandert hebt"

--mergen met de master
>git merge branchnaam master

>checkout master --> nu ben je in de master aant werken

>git push --> voert je veranderingen door op github dit is definitief!

--> als git push niet lukt:
>git pull

en dan opnieuw 

>git push
